# Active Directory Home Lab

## Overview

This project demonstrates a Windows Server 2022 Active Directory home lab built using VirtualBox.

## Technologies Used

- Windows Server 2022
- Active Directory Domain Services
- DNS
- PowerShell
- Windows 11
- VirtualBox

## Project Objectives

- Create and manage users
- Create Organizational Units (OUs)
- Configure DNS
- Join client computers to the domain
- Practice Active Directory administration
- Automate tasks with PowerShell

## Skills Demonstrated

- Active Directory Administration
- User and Group Management
- DNS Configuration
- Windows Server Administration
- PowerShell Automation
- Troubleshooting
